vi file_name ......... go into file (command mode), now you can read and copy&pase, but you can not write
i            ......... for insert mode, now you can write
esc          ......... go from insert mode to command mode
command mode:
	x .......... delete a charecter
	dd ......... delete entire line
	yy ......... copy a line
	p .......... pase
	CTRL + u ... page up
	CTRL + D ... page down
	: .......... take you to the prompt were you can type commands
	:w ......... save the file
